# Platform Features 🌟

## 1. Advanced Quiz System
*   **Adaptive Testing**: Challenges pulled from a live Firestore bank with difficulty scaling.
*   **Real-time Scoring**: Instant XP calculation and streak tracking.
*   **Pedagogical Feedback**: In-depth explanations for every answer choice.

## 2. Pro AI Study Assistant
*   **Gemini 1.5 Flash Integration**: High-speed, unlimited tutoring via BYOK model.
*   **Specialized Focus**: Strictly governed to discuss only JavaScript and Web technologies.
*   **Technical Rendering**: Support for Markdown and syntax-highlighted code blocks.

## 3. Interactive Code Lab
*   **Live Sandbox**: Browser-based JS execution environment.
*   **Snippet Library**: 30+ pre-built examples ranging from Closures to Bitwise operations.
*   **Console Inspection**: Real-time logging and return-value tracking.

## 4. Cybersecurity Academy
*   **Security Labs**: Hands-on training for XSS, Authentication security, and API protection.
*   **Vulnerability Simulations**: Practice defending against real-world attack vectors.

## 5. Gamification & Community
*   **Global Leaderboard**: Real-time ranking of users based on total XP.
*   **Progression Tracking**: Visual charts for XP growth and learning consistency.

## 6. Admin Command Center
*   **User Moderation**: Live oversight of all platform participants.
*   **Content Management (CMS)**: CRUD operations for the question bank.
*   **System Health**: Real-time platform metrics and security status logs.
